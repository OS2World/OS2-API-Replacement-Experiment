# OS2-API-Replacement-Experiment
This is an experiment on cloning the OS/2 API file by file with AI. 

This is a research experimient using AI to clone components of OS/2 Warp 4.52. 
The goals are:
- Produce C code and sometimes ASM for drivers.
- Compilable under OS/2 with the latest OpenWatcom compiler and linker

The source code is provided "AS IS".
